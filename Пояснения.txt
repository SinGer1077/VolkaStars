1. Для разработки использовался Haxe, фреймворк Kha. 
2. Графическое API - для отрисовки карты звезд kha.graphics4, для UI использовалась библиотека Zui и kha.graphics2.
3. Используемые ассеты - шрифт DroidSans.ttf для UI.
4. Для отображения звезд было решено использовать шейдер - full_galaxy.frag.glsl.
В нём отрисовываются как сами звезды, так и "свечение" галактики.
5. Средний фпс - 58
6. Описание классов:
	- Main - основной класс для запуска приложения и отрисовки всех составляющих.
	- GlobalState - статический класс для содержания общей информации о приложении (параметры мыши, клавиатуры, настройки звезд и т.д.).
	- IRenderClass - интерфейс, определяющий поведение объекта, который отрисовывается через kha.graphics4
	- Stars - класс для отрисовки карты звезд. Сначала формируется 2d меш в виде круга, затем на неё добавляется шейдер. Наследует IRenderClass.
	- MouseData - класс, отслеживающий управление мышью, содержащий функции OnMouseDOwn, OnMouseMove и т.д.
	- KeyboardData - класс, отслеживающий управление клавиатурой, содержащий функции OnKeyDown.
	- IUiClass - интерфейс, определяющий поведение объекта, который отрисовывается через kha.graphics2 и представляет собой элемент ui.
	- FPSCount - класс, отрисовывающий ui-текст, отображающий текущее кол-во фпс. Наследует IUiClass.
	- SeedButton - класс, отрисовывающий ui-кнопку, которая при нажатии изменяет сид генерации карты звезд. Наследует IUiClass.
	- StarsCountButtons - класс, отрисовывающий ui-кнопки, которые при нажатии изменяют кол-во отображаемых звезд. Наследует IUiClass.
	- StarsNames - класс, отрисовывающий ui-текст, который при максимальном приближении отображает "название" звезды, самой близкой к центру экрана. Наследует IUiClass.
	- StarsTypes - класс, отрисовывающий ui-чекбоксы, которые при нажатии регулируют отображаемые типы звезд. Наследует IUiClass.
